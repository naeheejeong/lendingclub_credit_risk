# LendingClub 대출 상환 예측 모델링

> ML · 리스크 예측 · 불균형 데이터

LendingClub 대출 데이터 기반 상환 가능성 예측 모델을 개발한 프로젝트

## 🎯 배경 · 문제정의
- 대출 리스크 관리에서 부도(미상환) 예측은 소수 클래스 — 불균형 데이터 문제
- 부도 고객을 놓치는 비용(FN)이 정상 고객 오분류 비용(FP)보다 훨씬 큼 → **Recall 중심 평가 설계**

## 🗂 데이터 · 방법
- LendingClub 대출 데이터 전처리 및 파생변수 설계
- 불균형 처리 기법 **3종 비교** (오버샘플링·언더샘플링·클래스 가중치)
- FN 비용을 고려한 평가 지표 설계, CatBoost 등 모델 비교

## 📈 결과 · 임팩트
- CatBoost · **Recall 0.65 · PR-AUC 0.90**
- 리스크 관리 관점에서 임계값 조정 근거와 비용-편익 해석 제시

## 🛠 Tech Stack
Python · scikit-learn · CatBoost · Pandas
